# веб-приложение для бронирования экскурсий с морской рыбалкой

Стек технологий: Java 21, Spring Boot, Hibernate, liquibase, H2, Thymeleaf, MongoDB

Курсовая работа, crud приложение с типами пользователей:
- Пользователь может:
      - Зарегистрироваться
      - Забронировать/отменить экскурсию
      - Поменять данные профиля
- Администратор может:
      - Создать/удалить экскурсию
      - Добавить/удалить лодку
      - Добавить/удалить тип рыб
      - Добавить/удалить место для рыбалки

Полностью реализован функционал для реляционной бд h2. Можно переключить реализацию на профиль для MongoDB, там реализован функционал для пользователя.
